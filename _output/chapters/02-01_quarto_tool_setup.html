Tool Setup

Your Complete Course Development Tool Guide

Introduction

Welcome to Module 0. Before beginning analytics, Quarto development, and programming workflows, students must install and configure:

  • Git & GitHub
  • Antigravity (Google’s VS Code–based IDE)
  • Quarto
  • Python
  • R
  • Pandoc
  • Graphviz

This module ensures every student has:

  • a working IDE
  • version control correctly configured
  • environments set up
  • the ability to edit, preview, and publish Quarto content
  • a GitHub Pages site (username.github.io)
  • experience with pull → edit → commit → push workflows

Learning Objectives

By the end of this module, you will:

  • Install Git, GitHub CLI, Quarto, Pandoc, Graphviz, Python 3.12, R, and RTools (Windows).
  • Install Antigravity and required plugins (Jupyter, Python, Quarto, Remote).
  • Configure Git + SSH authentication.
  • Create your GitHub Pages repository: username.github.io.
  • Clone, edit, commit, push, and pull changes.
  • Verify installation with screenshots and documentation.

Install Core System Tools

Antigravity requires certain system-level tools to function correctly.

Install Git

Git is the version control system used throughout this course.

Mac:

brew install git

Windows: Download and install: https://git-scm.com/download/win

Linux:

sudo apt update && sudo apt install git -y

Install GitHub CLI (gh)

This tool connects your IDE (Antigravity) with GitHub.

Check installation:

gh --version

Installation: https://cli.github.com/

Install Quarto

Used to build websites, slides, notebooks, and documents.

Download: https://quarto.org/docs/download/

Install Pandoc

Required for Quarto PDF and DOCX output.

Mac:

brew install pandoc

Linux:

sudo apt install pandoc -y

Install Graphviz

Required for diagrams, ML visualizations, and flowcharts.

Mac:

brew install graphviz

Linux:

sudo apt install graphviz -y

Install Python 3.12

Download: https://www.python.org/downloads/

Check: Add Python to PATH.

Linux:

sudo apt install python3 python3-pip -y

Install R

Download: https://cran.r-project.org/

Install RTools (Windows only)

Required to compile R packages.

https://cran.r-project.org/bin/windows/Rtools/

Download Antigravity

Configure Git Identity and SSH Keys

Set Git Username and Email

git config --global user.name "Your Name"
git config --global user.email "you@example.com"

Generate SSH Key

ssh-keygen -t ed25519

Press Enter to accept defaults.

Add Public Key to GitHub

cat ~/.ssh/id_ed25519.pub

Copy and paste it into GitHub: https://github.com/settings/keys

Test SSH Authentication

ssh -T git@github.com

Expected output:

Hi <username>! You've successfully authenticated.

Create Your GitHub Pages Repository

This repository will host your personal website and teach you core Git workflows.

Create Repository on GitHub

Go to: https://github.com/new

Repository name:

yourusername.github.io

Select:

  • Public
  • Add README
  • Add .gitignore (Python or R)

Clone the Repository Locally

git clone git@github.com:yourusername/yourusername.github.io.git
cd yourusername.github.io

Make Your First File

echo "Welcome to my GitHub website!" >> index.md

Commit Your Change

git add .
git commit -m "Initial website content"

Push to GitHub

git push origin main

Pull Changes to Confirm Workflow

git pull

Python and R Environment Setup

You may use Python, R, or both inside Antigravity.

Create Python Virtual Environment

python3 -m venv .venv

Activate:

Mac/Linux:

source .venv/bin/activate

Windows:

.venv\Scripts\activate

Install common packages:

pip install numpy pandas jupyter matplotlib quarto

Initialize R Environment

Open R and run:

install.packages("renv")
renv::init()
install.packages(c("tidyverse", "knitr", "rmarkdown"))
renv::snapshot()
